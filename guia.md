# Manual do 1 Ano Técnico
# Organização: não deixe tudo para a última hora

Aprenda a se organizar desde o primeiro dia.

Como estudante da ETEC, um dos maiores erros que cometi foi acreditar que teria tempo para fazer tudo depois. No começo parece que dá para deixar um trabalho para amanhã ou estudar para uma prova na véspera, mas, conforme as semanas passam, as atividades começam a se acumular. Quando você percebe, está  virando madrugadas para entregar vários trabalhos ao mesmo tempo e ainda estudar para provas importantes.

Criar o hábito de se adiantar faz uma diferença enorme. Tanto para a escola quanto para a vida. Assim que um trabalho for passado, reserve um tempo para começar, mesmo que seja apenas pesquisando o assunto ou montando a estrutura. Da mesma forma, não espere um dia antes da prova para estudar. Revisar a matéria aos poucos ajuda bastante e é bem melhor do que tentar aprender tudo em uma única noite. PRINCIPALMENTE as matérias do curso, progamação, não é algo que se aprende em horas.

Usar uma agenda, ou um aplicativo de notas do celular pode ajudar a organizar prazos e evitar esquecimentos. Quando você planeja seu tempo, não significa que vai gastar o dia inteiro só fazendo aquilo, mas sim distribuir as tarefas de forma equilibrada para que elas não se tornem um peso.
